## 概述

### 大英图书馆简介

**大英图书馆** (British Library) 是世界上最大的图书馆之一：
- **位置**: 伦敦，英国
- **成立时间**: 1973 年 (源于大英博物馆图书馆)
- **馆藏**: 1.7 亿 + 件物品
- **蒙古文收藏**: 亚洲与非洲研究部

### 蒙古文收藏规模

| 类型 | 数量 | 年代 |
|------|------|------|
| 手稿 | ~500 件 | 18-20 世纪 |
| 木刻本 | ~300 件 | 19-20 世纪 |
| 地图 | ~50 件 | 19-20 世纪 |
| 档案 | ~200 件 | 20 世纪 |
| **总计** | **~1,050 件** | **-** |

**数字化状态**:
- ✅ 已数字化：~200 件 (20%)
- ⏳ 进行中：~100 件
- ❌ 未数字化：~750 件 (70%)

## IIIF 集成

### 什么是 IIIF

**IIIF** (International Image Interoperability Framework):
- 图像互操作框架
- 标准化 API
- 支持深度缩放、标注、比较

### 大英图书馆 IIIF API

**Manifest URL 格式**:
```
https://api.bl.uk/metadata/iiif/{ark_id}/manifest.json
```

**示例** (蒙古秘史):
```json
{
  "@context": "http://iiif.io/api/presentation/2/context.json",
  "@type": "sc:Manifest",
  "@id": "https://api.bl.uk/metadata/iiif/ark:/81055/vdc_100000000123/manifest.json",
  "label": "Secret History of the Mongols",
  "sequences": [
    {
      "@type": "sc:Sequence",
      "canvases": [
        {
          "@id": "https://api.bl.uk/metadata/iiif/ark:/81055/vdc_100000000123/canvas/p1",
          "@type": "sc:Canvas",
          "label": "Page 1",
          "images": [
            {
              "@type": "oa:Annotation",
              "resource": {
                "@id": "https://api.bl.uk/image/iiif/ark:/81055/vdc_100000000123/full/full/0/default.jpg",
                "@type": "dctypes:Image",
                "format": "image/jpeg",
                "width": 3000,
                "height": 4000
              }
            }
          ]
        }
      ]
    }
  ]
}
```

### 使用示例

#### 1. Mirador 查看器

```html
<!-- 嵌入 Mirador 查看器 -->
<div id="mirador"></div>
<script src="https://unpkg.com/mirador@3/dist/mirador.min.js"></script>
<script>
  const miradorInstance = Mirador.viewer({
    id: 'mirador',
    windows: [{
      loadedManifest: 'https://api.bl.uk/metadata/iiif/ark:/81055/vdc_100000000123/manifest.json'
    }]
  });
</script>
```

#### 2. 程序化访问

```python
import requests

# 获取 Manifest
manifest_url = "https://api.bl.uk/metadata/iiif/ark:/81055/vdc_100000000123/manifest.json"
response = requests.get(manifest_url)
manifest = response.json()

# 提取图像 URL
for sequence in manifest['sequences']:
    for canvas in sequence['canvases']:
        image_url = canvas['images'][0]['resource']['@id']
        print(f"Image: {image_url}")
```

#### 3. 下载高分辨率图像

```python
# IIIF Image API
base_url = "https://api.bl.uk/image/iiif/ark:/81055/vdc_100000000123"

# 请求特定区域和尺寸
params = {
    'region': 'full',
    'size': '1000,',  # 宽度 1000px
    'rotation': '0',
    'quality': 'default',
    'format': 'jpg'
}

image_url = f"{base_url}/{params['region']}/{params['size']}/{params['rotation']}/{params['quality']}.{params['format']}"
response = requests.get(image_url)

with open('mongolian_manuscript.jpg', 'wb') as f:
    f.write(response.content)
```

## 引用指南

### 手稿引用格式

```
British Library, Or. [编号], [描述], [年代].
```

**示例**:
```
British Library, Or. 6803, Secret History of the Mongols, 17th century.
```

### 数字化藏品引用

```
British Library, "[标题]", [年代], [编号], 
https://www.bl.uk/manuscripts/FullDisplay.aspx?ref=[编号], accessed [日期].
```

**示例**:
```
British Library, "Secret History of the Mongols", 17th century, Or. 6803,
https://www.bl.uk/manuscripts/FullDisplay.aspx?ref=Or_6803, accessed 2026-04-03.
```

### IIIF 图像引用

```
British Library, IIIF Manifest: [ark_id], 
https://api.bl.uk/metadata/iiif/[ark_id]/manifest.json
```
