- Python 3.8+
- 无特殊硬件要求

## 使用示例

### 基础对话

```bash
$ python3 main.py

📝 您：ᠰᠠᠶᠢᠨ ᠪᠠᠶᠢᠨᠤᠤ
🤖 AI: ᠰᠠᠶᠢᠨ ᠪᠠᠶᠢᠨᠤᠤ！ᠪᠢ ᠮᠣᠩᠭᠣᠯ AI ᠲᠠᠨᠢᠯᠴᠠᠭᠤᠯᠠᠭᠴᠢ ᠪᠠᠶᠢᠨᠠ᠃
     你好！我是蒙古 AI 助手。有什么可以帮您？
```

### 翻译功能

```bash
$ python3 main.py --translate "ᠪᠠᠶᠠᠷᠲᠠᠢ"
原文：ᠪᠠᠶᠠᠷᠲᠠᠢ
翻译：谢谢
```

### 文本分析

```bash
$ python3 main.py --text "ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ"
ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ (蒙古文)
  包含蒙古文：✅
  蒙古文内容：ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ
  转写：moᠩgol bičig
```

## 已知问题

1. **词汇量有限** - 目前只有 20 个常用词
2. **翻译简单** - 仅支持单词级别翻译
3. **对话简单** - 预定义回复，非真正 AI 对话
4. **无上下文理解** - 不支持多轮对话上下文
