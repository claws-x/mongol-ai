## 🆕 新增功能

### 1. 语音输入支持 🎤

**用户说话，AI 文字回复**

- ✅ macOS 系统听写集成
- ✅ 自动语音识别（可选）
- ✅ 降级方案：语音不可用时自动切文字
- ✅ 支持中英文语音输入

**使用方式**:
```bash
# 语音模式
python3 main_voice.py --voice

# 文字模式（默认）
python3 main_voice.py
```

---

### 2. 简化的语音输入流程

**无需安装任何依赖即可使用**:

1. 运行 `python3 main_voice.py --voice`
2. 看到提示后按 `Fn` 键两次
3. 说话（macOS 自动转文字）
4. AI 用蒙古文 + 中文回复

### 场景 1: 学习蒙古文

```

AI (文字):  ᠰᠠᠶᠢᠨ ᠪᠠᠶᠢᠨᠤᠤ

### 场景 2: 练习发音

```
用户 (语音): "谢谢怎么说？"
AI (文字):  ᠪᠠᠶᠠᠷᠯᠠᠯ᠎ᠠ
           谢谢！发音："bayartai"
```

### 场景 3: 日常对话

```
用户 (语音): "今天天气怎么样？"

## ⚠️ 已知限制

1. **蒙古语语音识别**: 支持有限，建议用中文说"蒙古文怎么说"
2. **环境噪音**: 在安静环境使用效果更佳
3. **网络依赖**: Google API 需要网络（简化版不需要）
