# 🎤 语音输入设置指南

## 功能说明

**语音输入版**允许您用**说话**的方式与 AI 助手交流，AI 用**文字**回复。

---

## 快速开始

### 方式 1: 简化版（推荐 - 无需安装）

使用 macOS 系统听写功能：

```bash
python3 main_voice.py --voice
```

**使用方法**:
1. 运行程序后，看到 `🎤 您 (语音/文字):` 提示
2. 按 `Fn` 键两次（或 `Ctrl+Ctrl`）启动 macOS 听写
3. 对着麦克风说话
4. 系统会自动将语音转为文字输入
5. AI 用蒙古文 + 中文回复

---

### 方式 2: 完整版（需要安装依赖）

安装语音识别库：

```bash
# 安装依赖
pip3 install SpeechRecognition pyaudio

# macOS 还需要安装 portaudio
brew install portaudio
```

然后运行：

```bash
python3 main_voice.py --voice
```

## 支持的输入方式

| 方式 | 命令 | 说明 |
|------|------|------|
| **语音输入** | `python3 main_voice.py --voice` | 说话 → 文字识别 |
| **文字输入** | `python3 main_voice.py` | 键盘输入 |
| **单次对话** | `python3 main_voice.py --text "你好"` | 一次性对话 |

---

## 语音命令

在对话中可以输入：

| 命令 | 功能 |
|------|------|
| `/voice` | 切换语音/文字模式 |
| `/help` | 查看帮助 |
| `/clear` | 清空对话历史 |
| `/quit` | 退出程序 |

## 故障排除

### 问题 1: 语音模块未加载

**症状**: `⚠️ 语音模块未加载，将使用文字输入`

**解决**: 
- 这是正常的，程序会降级到文字输入
- 如需语音功能，安装依赖：`pip3 install SpeechRecognition pyaudio`

### 问题 2: macOS 听写无法使用

**症状**: 按 Fn 键没反应

**解决**:
1. 打开 系统设置 → 键盘 → 听写
2. 确保"听写"已开启
3. 设置听写快捷键（默认 Fn 键两次）

### 问题 3: PyAudio 安装失败

**症状**: `error: command 'gcc' failed`

**解决**:
```bash
# macOS
brew install portaudio
pip3 install pyaudio

# 或使用 conda
conda install pyaudio
```

| **英文** | ✅ 完全支持 | 使用英语 |

## 最佳实践

1. **安静环境**: 在安静的地方使用语音输入
2. **清晰发音**: 说话要清晰、缓慢
3. **靠近麦克风**: 距离麦克风 10-20 厘米
4. **短句输入**: 每次说一句话，不要过长
