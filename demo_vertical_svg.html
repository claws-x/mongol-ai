✨ SVG 方案优势

✅ 高质量渲染

SVG 矢量图形，无损缩放，适合高质量展示和打印

✅ 精确控制

精确控制每个字符的位置、大小和样式

✅ 不依赖 CSS

SVG 原生支持 writing-mode，不依赖浏览器 CSS 实现

⚠️ 适用场景

适合静态展示、Logo、标题，不适合输入界面
