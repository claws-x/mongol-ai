## 📱 嵌入式输入法

### 功能特点

1. **独立页面** - 可在任何网站嵌入使用
2. **虚拟键盘** - 5 个分类键盘（元音/辅音 1/辅音 2/辅音 3/功能）
3. **智能建议** - 8 个常用短语随机显示
4. **实时统计** - 字符数和词数统计
5. **工具按钮** - 清空/复制/粘贴/示例

### 使用方式

**独立使用**：
```bash
# 直接在浏览器打开
open embedded-input.html
```

**嵌入网页**：
```html
<iframe 
    src="embedded-input.html" 
    width="800" 
    height="600"
    frameborder="0">
</iframe>
```
