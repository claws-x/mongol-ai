├────┤

├────┤
│ ᠢ  │ ← I (i)

## 字符显示方向

### 蒙古文字符
- **显示方式**: `transform: rotate(90deg)`
- **原因**: 蒙古文字母在字体中默认横向显示
- **效果**: 旋转 90 度后呈现竖排效果

### 中文字符
- **显示方式**: `transform: rotate(-90deg)`
- **原因**: 中文需要向左旋转 90 度竖排
- **应用**: "空格"、"发送"

### 英文字符
- **显示方式**: `transform: rotate(-90deg)`
- **原因**: 英文需要向左旋转 90 度竖排
- **应用**: "Space"、"Send"

## 颜色方案

### 主题色
- **AI 渐变**: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- **主色**: `#007AFF` (蓝色)
