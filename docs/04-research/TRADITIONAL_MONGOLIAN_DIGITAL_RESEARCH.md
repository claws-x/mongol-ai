## 📚 第一部分：传统蒙古文文字特性研究

### 1.1 文字历史

**起源**：
- 公元 1204 年，成吉思汗采用回鹘式蒙古文
- 800 多年历史，世界上最古老的竖排文字之一
- 从左到右书写，从上到下换行

**特点**：
- **竖排书写** - 世界上少数竖排文字
- **连写特性** - 词内字符连写
- **位置变体** - 词首/词中/词尾不同形态
- **音素文字** - 一个字符代表一个音素

### 1.2 Unicode 编码标准

**Unicode 范围**：
```
U+1800 - U+18AF  蒙古文区块 (Mongolian)
U+11660 - U+1167F 蒙古文补充 (Mongolian Supplement)
```

**编码问题**：
1. **字序 vs 显示序** - Unicode 存储顺序与显示顺序不同
2. **自由变体选择符** - FVS1/FVS2/FVS3 控制字符形态
3. **零宽连接符** - ZWJ/ZWNJ 控制连写
4. **NFC vs NFD** - 规范化形式影响显示

### 1.3 竖排显示技术挑战

**CSS 方案对比**：

```css
/* 方案 1: writing-mode (标准方案) */
.mongolian {
    writing-mode: vertical-lr;
    text-orientation: upright;
}
/* ✅ 优点：W3C 标准，浏览器原生支持 */
/* ❌ 缺点：部分老浏览器不支持 */

/* 方案 2: transform 旋转 */
.mongolian {
    transform: rotate(90deg);
    transform-origin: center center;
}
/* ✅ 优点：兼容性好，所有浏览器支持 */
/* ❌ 缺点：不是真正的竖排，是旋转 */

/* 方案 3: SVG 竖排 */
<text writing-mode="tb" direction="rtl">蒙古文</text>
/* ✅ 优点：精确控制，SVG 原生支持 */
/* ❌ 缺点：需要 SVG，文本处理复杂 */
```

**推荐方案**：
```css
.mongolian-vertical {
    writing-mode: vertical-lr !important;
    text-orientation: upright !important;
    -webkit-writing-mode: vertical-lr !important;
    -webkit-text-orientation: upright !important;
    
    /* 兼容方案：如果 writing-mode 不支持，使用旋转 */
    transform: rotate(90deg);
    transform-origin: center center;
}
```

### 2.5 文字识别 (OCR)

**传统蒙古文 OCR 挑战**：
1. **竖排文本** - 大多数 OCR 引擎针对横排
2. **连写特性** - 字符连写增加识别难度
3. **位置变体** - 同一字符不同位置不同形态
4. **训练数据** - 缺乏标注数据

**解决方案**：
```python
# 基于深度学习的 OCR
class MongolianOCR:
    def __init__(self):
        # 使用 CRNN 或 Transformer
        self.model = load_model('mongolian_ocr.pth')
    
    def recognize(self, image):
        # 1. 图像预处理（旋转 90 度）
        image = rotate(image, -90)
        
        # 2. 文本检测
        text_regions = detect_text(image)
        
        # 3. 文字识别
        texts = []
        for region in text_regions:
            text = recognize_text(region)
            texts.append(text)
        
        return texts
```
