## 🏢 历史发展

### 1999-2005: 创业期
- 1999: 公司成立，开发首款蒙古文输入法
- 2001: 发布 Menksoft IME 1.0 (Windows 98/2000)
- 2003: 推出首套商业蒙古文字库
- 2005: 占据内蒙古市场 80% 份额

### 2006-2012: 扩张期
- 2006: 进入蒙古国市场
- 2008: 发布 Menksoft Office (蒙古文办公软件)
- 2010: 推出蒙古文 OCR 系统
- 2012: 用户突破 100 万

### 2013-2019: 成熟期
- 2013: 发布 Menksoft Mobile (Android/iOS)
- 2015: 推出蒙古文语音识别
- 2017: 与微软合作 Windows 蒙古文支持
- 2019: 用户突破 500 万

### 2020-至今：转型期
- 2020: 疫情影响，线下业务受挫
- 2021: 转向云服务与 SaaS
- 2023: 推出 AI 驱动的蒙古文 NLP

## 🔍 SWOT 分析

### 优势 (Strengths)
1. **市场先发优势**: 25 年积累，用户习惯固化
2. **政府关系**: 内蒙古政府采购供应商
3. **完整产品线**: 输入法 + 字库 + 办公 + OCR + 云
4. **本地化深度**: 符合中国国家标准
5. **词库资源**: 50 万 + 专业词条
6. **渠道网络**: 线下经销商覆盖内蒙古全境

### 劣势 (Weaknesses)
1. **技术落后**: OCR/NLP 仍用传统方法
2. **闭源模式**: 无法利用社区力量
3. **竖排支持弱**: 不符合蒙古文传统
4. **移动端体验差**: 输入/显示优化不足
5. **价格高**: 个人用户负担重
6. **国际化弱**: 英文文档缺失

### 机会 (Opportunities)
1. **AI 转型**: 引入深度学习提升 OCR/NLP
2. **云服务**: SaaS 模式扩大收入
3. **蒙古国市场**: 政治关系改善
4. **教育市场**: 学校数字化需求
5. **文化遗产**: 古籍数字化项目

### 威胁 (Threats)
1. **开源竞争**: tugstugi 等项目免费优质
2. **技术颠覆**: AI 大模型降低门槛
3. **用户流失**: 年轻一代偏好开源
4. **政策风险**: 政府采购政策变化
5. **国际巨头**: Google/Microsoft 进入

## 🎯 本项目差异化策略

### 1. 竖排优先 (核心差异)
**Menksoft**: 横排为主，竖排为附加功能  
**本项目**: 竖排为默认，所有组件原生支持

```css
/* 本项目默认样式 */
.mongolian-text {
    writing-mode: vertical-lr !important;
    text-orientation: upright !important;
}

/* Menksoft 需要手动切换 */
.menksoft-vertical {
    writing-mode: vertical-lr; /* 非默认 */
}
```

### 2. 移动优先
**Menksoft**: 桌面端优先，移动端适配  
**本项目**: 移动端优先，桌面端增强

| 场景 | Menksoft | 本项目 |
|------|----------|--------|
| 手机输入框 | 横排 | 竖排 SVG 降级 |
| 触摸键盘 | 标准布局 | 竖排优化布局 |
| 小屏渲染 | 缩放 | 自适应布局 |
| 离线支持 | 有限 | Service Worker |

### 3. Web 原生
**Menksoft**: Windows 桌面应用为主  
**本项目**: HTML5 + Python 后端

| 特性 | Menksoft | 本项目 |
|------|----------|--------|
| 安装 | 需要安装包 | 浏览器访问 |
| 更新 | 手动升级 | 自动更新 |
| 跨平台 | Windows 为主 | 全平台 |
| 集成 | 本地 API | RESTful API |

### 4. 开源开放
**Menksoft**: 闭源商业，禁止分发  
**本项目**: MIT/Apache/OFL/CC BY

| 方面 | Menksoft | 本项目 |
|------|----------|--------|
| 代码 | ❌ 闭源 | ✅ MIT/Apache |
| 字体 | ❌ 商业授权 | ✅ OFL |
| 文档 | ❌ 付费 | ✅ CC BY 4.0 |
| 数据 | ❌ 专有 | ✅ CC BY-SA |
| 社区 | ❌ 无贡献渠道 | ✅ GitHub PR |

### 5. AI 驱动
**Menksoft**: 传统方法 + 有限 AI  
**本项目**: 深度学习优先

| 任务 | Menksoft | 本项目 |
|------|----------|--------|
| OCR | HMM + 模板 | Transformer |
| NLP | 规则 + 统计 | BERT/GPT |
| ASR | GMM-HMM | wav2vec2 |
| 翻译 | 统计 MT | Neural MT |
