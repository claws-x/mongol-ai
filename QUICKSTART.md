# 🚀 快速开始指南

## 安装

```bash
cd mongolian-ai-assistant
pip3 install -r requirements.txt
```

---

## 使用方法

### 1. 交互模式（推荐）

```bash
python3 main.py
```

示例对话：
```
📝 您：ᠰᠠᠶᠢᠨ ᠪᠠᠶᠢᠨᠤᠤ
🤖 AI: ᠰᠠᠶᠢᠨ ᠪᠠᠶᠢᠨᠤᠤ！ᠪᠢ ᠮᠣᠩᠭᠣᠯ AI ᠲᠠᠨᠢᠯᠴᠠᠭᠤᠯᠠᠭᠴᠢ ᠪᠠᠶᠢᠨᠠ᠃
     你好！我是蒙古 AI 助手。

📝 您：你好
🤖 AI: ᠰᠠᠶᠢᠨ ᠪᠠᠶᠢᠨᠤᠤ！...
     你好！...
```

### 2. 单次对话

```bash
python3 main.py --text "ᠰᠠᠶᠢᠨ ᠪᠠᠶᠢᠨᠤᠤ"
```

### 3. 翻译模式

```bash
# 蒙古文翻译
python3 main.py --translate "ᠪᠠᠶᠠᠷᠯᠠᠯ᠎ᠠ"

# 中文翻译
python3 main.py --translate "你好" --from zh --to mn
```

## 常用命令

| 命令 | 功能 |
|------|------|
| `/help` | 查看帮助 |
| `/clear` | 清空对话历史 |
| `/history` | 查看对话历史 |
| `/analyze` | 分析文本 |
| `/quit` | 退出程序 |
