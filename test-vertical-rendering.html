测试 5: 浏览器兼容性
标准 CSS 属性
.mongolian-vertical {
    writing-mode: vertical-lr;           /* 竖排，从左到右 */
    -webkit-writing-mode: vertical-lr;   /* Safari/Chrome */
    -moz-writing-mode: vertical-lr;      /* Firefox */
    -o-writing-mode: vertical-lr;        /* Opera */
    -ms-writing-mode: tb-lr;             /* IE */
    
    text-orientation: upright;           /* 字符直立 */
    -webkit-text-orientation: upright;   /* Safari/Chrome */
    
    font-family: 'Mongolian Baiti', sans-serif;
    font-size: 24px;
}
✅ 支持：Chrome 48+, Firefox 41+, Safari 12+, Edge 79+
✅ 结论 - 蒙古文竖排正确显示方案

推荐方案：

.mongolian-text {
    writing-mode: vertical-lr;           /* 关键：竖排 + 从左到右 */
    text-orientation: upright;           /* 关键：字符直立 */
    -webkit-writing-mode: vertical-lr;   /* Safari 兼容 */
    -webkit-text-orientation: upright;   /* Safari 兼容 */
    
    font-family: 'Mongolian Baiti', 'Noto Sans Mongolian', sans-serif;
    font-size: 24px;
    line-height: 1.5;
}

关键点：

  • writing-mode: vertical-lr - 竖排 + 从左到右换列
  • text-orientation: upright - 字符保持直立方向
  • 不要使用 transform: rotate() - 这是临时方案，不推荐
  • 使用标准 CSS 属性 - 现代浏览器原生支持
✅ 这是 W3C 标准方案，所有现代浏览器都支持
